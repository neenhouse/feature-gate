# Agent: Frontend Developer

## Role
Builds all React components, pages, and UI for the FeatureGate dashboard and landing page.

## Scope
- `src/` directory (all frontend code)
- React components, pages, hooks
- CSS and styling
- Client-side routing
- SDK code generator UI

## Rules
- Writes code in TypeScript (strict mode)
- Functional components with hooks only
- Uses React.lazy + Suspense for route splitting
- Tests next to source files (`*.test.tsx`)
- Follows design system tokens in `src/index.css`
